# üìä AI MATCHING TECHNICAL SUMMARY
*Use this for reference when explaining the algorithm*

## How the Match Score is Calculated:

```
MATCH SCORE = 
    (Topic Overlap √ó 0.40)     ‚Üê 40% weight - most important
  + (Language Match √ó 0.20)    ‚Üê 20% weight
  + (Culture Match √ó 0.15)     ‚Üê 15% weight
  + (Support Style √ó 0.15)     ‚Üê 15% weight
  + (Academic Context √ó 0.10)  ‚Üê 10% weight (grad year, degree)
```

## Profile Dimensions the AI Analyzes:

| Dimension | What It Captures | Weight |
|-----------|------------------|--------|
| Support Topics | Anxiety, depression, stress, relationships, etc. | 40% |
| Languages | English, Spanish, Hindi, Mandarin, etc. | 20% |
| Cultural Background | East Asia, South Asia, Middle East, Europe, etc. | 15% |
| Support Style | Listening, Advice, Mixed | 15% |
| Academic Context | Freshman-Senior, Bachelor's-PhD, Graduation Year | 10% |

## Semantic Embedding Explanation (Simple Terms):

> "The AI converts your entire profile into a 384-dimensional vector - essentially, a unique mathematical fingerprint. When comparing two users, the AI calculates the cosine similarity between their vectors. A score closer to 1.0 means near-identical profiles; closer to 0 means no overlap. We convert this to a percentage for easy understanding."
